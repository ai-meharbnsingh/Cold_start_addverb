# Integration Guide

This guide covers the integration process for the Cold Start project.

## Installation

1. Clone the repository: `git clone https://github.com/ai-meharbnsingh/Cold_start_addverb.git`
2. Navigate to the directory: `cd Cold_start_addverb`
3. Run the installation script: `npm install`

## Usage

To use the application, run the following command:

```bash
npm start
```
